# Cumulative RSI 전략 분석 결과

## 📚 Cumulative RSI란?

**출처:** Larry Connors, "Cumulative RSI 전략"

**개념:**
```python
# 기존 RSI(2): 단일 시점의 RSI 값
rsi_2 = RSI(close, 2)

# Cumulative RSI: N일간 RSI(2) 합계
cumulative_rsi_2days = rsi_today + rsi_yesterday
cumulative_rsi_3days = rsi_today + rsi_yesterday + rsi_day_before
```

**원리:**
- 더 극단적인 과매도 상태만 포착
- 노이즈 감소, 신호 품질 향상
- 미국 시장 백테스트: MDD 57% → 37% 개선

## 🎯 핵심 발견

### 1. Cumulative RSI의 효과

**장점:**
- ✅ MDD 대폭 개선: -55% → -38% (단독), -12% (진입 강화 시)
- ✅ 위험조정수익 향상
- ✅ 더 안정적인 수익 곡선

**단점:**
- ⚠️ 수익률 감소: 21.71% → 16.19% (-25%)
- ⚠️ 거래 빈도 증가: 412회 → 522회 (+27%)

### 2. 놀라운 조합: Cumulative RSI + 진입 강화

```python
CumulativeRSIEngine(
    cumulative_days=2,
    cumulative_buy_threshold=10,
    price_drop_threshold=-5.0,  # 진입 강화
)
```

**결과:**
- 연수익률: 10.98%
- **MDD: -12.10%** 🎉 (기존 -55%의 1/5 수준!)
- 위험조정수익: 0.9075 (2배 이상 개선)

**의미:**
- 안정성 최우선 투자자에게 최적
- 은행 정기예금(연 3%) 대비 3.6배 수익
- MDD -12%는 대부분의 투자자가 견딜 수 있는 수준

### 3. 전략별 비교

| 전략 | 수익률 | MDD | 특징 | 적합 투자자 |
|-----|--------|-----|------|-----------|
| 기존 RSI + 현금20% + 진입강화 | 25.53% | -49.35% | 최고 수익 | 공격적 |
| Cumulative + 진입강화 | 10.98% | **-12.10%** | **최소 MDD** | **보수적** ✨ |
| Cumulative 단독 | 16.19% | -38.49% | 균형 | 중도적 |

## 🎓 결론 및 권장사항

### 1. Cumulative RSI는 효과적인가?

**Yes, but...**
- ✅ MDD 개선 효과 확실 (-55% → -38%)
- ✅ 위험조정수익 향상
- ⚠️ 수익률 감소는 trade-off

### 2. 최적 전략은?

**투자 성향에 따라:**

**A. 수익률 최우선 (공격적)**
→ 기존 RSI + 현금20% + 진입강화
- 연 25.53%, MDD -49%

**B. 안정성 최우선 (보수적)** ⭐
→ **Cumulative RSI + 진입강화**
- 연 10.98%, **MDD -12%**
- 가장 추천!

**C. 균형형**
→ Cumulative RSI 단독
- 연 16.54%, MDD -39%

### 3. 한국 시장 특수성

미국 시장(원 논문):
- 기존 RSI: 연 26.8%, MDD -57%
- Cumulative: 연 26.6%, MDD -37%
- 수익률 유지하면서 MDD 개선

한국 시장(우리 테스트):
- 기존 RSI: 연 21.71%, MDD -55%
- Cumulative: 연 16.19%, MDD -38%
- **수익률 감소가 더 큼**

**이유 추정:**
1. 한국 시장 변동성이 더 큼
2. 종목 수 제한 (100개)
3. 거래 기회 감소 효과 더 큼
