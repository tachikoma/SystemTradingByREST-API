### 실험별 상세

#### ① SIGNAL_STR — 신호 강도 포지셔닝
- **방식**: 신호 강도(`(70 - RSI) / 70`)^2 배율로 현금 사용 비중 가변 할당
- **결과**: **+40.03%** — baseline(-47.96%) 대비 -7.93%p 열위
- **분석**: 신호가 강할수록 더 많이 베팅하는 방식이 오히려 평균 수익률을 낮춤. 포지션 크기를 균등하게 유지하는 것이 이 전략에서 더 효과적.

#### ② RSI_CROSS — RSI 교차 매도
- **방식**: RSI(2)가 70 초과 후 50 아래로 내려오면 매도 (trailing stop 형태)
- **결과**: **-38.50%**, Sharpe -0.311 — **완전 실패**
- **분석**: 
  - 70 위에서 50 아래로의 하락은 이미 상당한 조정이 진행된 상태 → 수익을 거둘 기회를 놓침
  - 거래 수 절반 감소(1,662 → 940), 승률 80.27% → 67.74% 급락
  - RSI 교차 방식은 RSI(2) 역추세 전략과 궁합이 맞지 않음

#### ③ VOLUME_FILTER — 거래량 필터
- **방식**: 매수 신호 발생 시 당일 예상 거래량(volume × 20)을 MA20×50%와 비교, 미만이면 매수 스킵
- **결과**: **+12.87%** — baseline 대비 -35.09%p 저조
- **분석**:
  - 거래가 1,662 → 1,510으로 감소 (필터에 탈락한 종목들이 오히려 반등)
  - 역추세 매수 시점은 대부분 거래량이 낮은 상태 — 거래량 필터가 오히려 가장 좋은 진입 기회를 차단
  - MDD는 -33.29% → -29.47%로 개선되었으나 수익성 하락폭이 너무 큼
  - **차후 재검토**: 0.5배가 아닌 0.2~0.3배 같은 덜 제한적인 기준으로 추가 실험 가능

#### ④ ATR_TRAIL_STOP — ATR 트레일링 스탑 (유일하게 근소 우위)
- **방식**: 매도 조건 (`RSI>70 + breakeven`)에 **ATR(14) 기반 트레일링 스탑** 추가. 수익률이 +5% 이상이면 최고가 대비 2×ATR 하락 시 매도. -5% 미만이거나 breakeven 미만이면 활성화 안 함.
- **결과**: **+49.33%** (baseline +47.96% 대비 +1.37%p), Sharpe 0.365 (baseline 0.359 대비 +1.7%)
- **분석**:
  - 거래 수 1,662 → 1,716 (+54건, +3.2%) — ATR 스탑이 RSI 조건보다 먼저 발동되는 사례 발생
  - 평균 수익률 1.44% → 1.46% 소폭 상승
  - MDD -33.29% → -33.09%, 거의 동일
  - **개선이 미미(≈1.4%p)하여 baseline 대비 확실한 우위라고 보기 어려움**
  - 단, MDD를 유지하면서 거래 건수를 늘린 점은 긍정적

### 결론

- **baseline (TSL180_PT0_SELL70_NO_MA20, +47.96%)이 여전히 최상위 성과**
- **ATR 트레일링 스탑**이 유일하게 baseline 이상의 총수익률(+49.33%)을 보였으나 개선 폭이 1.37%p로 미미
- 신호 강도 포지셔닝(-7.93%p), 거래량 필터(-35.09%p), RSI 교차 매도(-86.46%p)는 모두 baseline보다 저조
- **권장**: 추가 실험 없이 현재 baseline 유지. ATR 트레일링 스탑은 선택적 옵션으로 engine에 남겨두되 기본값은 사용 안 함.

   - ADX 임계값을 30~35로 높이면 더 나을 수 있음
