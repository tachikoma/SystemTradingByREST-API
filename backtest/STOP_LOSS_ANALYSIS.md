### 2. 왜 손절이 효과 없는가?

**RSI(2) 역추세 전략의 특성:**

```
전략 로직: 급락 → 매수 → 반등 → 매도

실제 프로세스:
1. 주가 급락 (-10~30%)
2. RSI < 5 → 매수
3. 일시적 추가 하락 가능 (-5~20% 더)  ← 여기서 손절하면 손실
4. 반등 시작
5. RSI > 80 → 매도 (평균 수익 7.55%)

손절 효과:
- 가격 손절: 반등 전에 자름 → 수익 기회 상실
- 시간 손절: 긴 조정 기간을 견디지 못함
```

### 3. 구체적 사례
