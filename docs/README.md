# 키움 REST API 문서 (Markdown)

## 📚 개요

키움증권 REST API 엑셀 문서를 AI가 쉽게 읽고 검색할 수 있는 마크다운 형식으로 변환한 문서입니다.

- **원본**: `키움 REST API 문서.xlsx`
- **총 API 수**: 206개
- **생성일**: 2026-01-03

## 🎯 용도

이 마크다운 문서는 다음과 같은 목적으로 사용됩니다:

1. **AI 에이전트 참조**: GitHub Copilot, ChatGPT 등이 API 스펙을 쉽게 검색하고 이해
2. **빠른 검색**: 텍스트 기반 검색으로 엑셀보다 빠른 API 탐색
3. **버전 관리**: Git으로 API 문서 변경 이력 추적
4. **웹 브라우저**: GitHub에서 바로 읽을 수 있는 포맷

## 📁 문서 구조

```
docs/
├── README.md                    # 이 파일
└── kiwoom_api/
    ├── README.md                # API 전체 목차 + 리스트
    ├── ERROR_CODES.md           # 오류 코드 목록
    ├── 접근토큰 발급(au10001).md
    ├── 주식기본정보요청(ka10001).md
    └── ... (206개 API 문서)
```

### 파일 명명 규칙

- `{API명}({API_ID}).md` 형식
- 예: `접근토큰 발급(au10001).md`, `주식 매수주문(kt10000).md`

## 📖 사용 방법

### 1. 전체 API 목록 확인

[kiwoom_api/README.md](kiwoom_api/README.md)에서 카테고리별 전체 API 목록을 확인할 수 있습니다.

### 2. 특정 API 검색

**VS Code에서 검색:**
```bash
# 파일명으로 검색 (Cmd/Ctrl + P)
주식 매수

# 내용으로 검색 (Cmd/Ctrl + Shift + F)
authorization
```

**GitHub에서 검색:**
- Repository 페이지에서 `t` 키를 눌러 파일 검색
- 또는 `s` 키를 눌러 전체 코드 검색

### 3. AI에게 API 문서 제공

AI 채팅에서 다음과 같이 사용할 수 있습니다:

```
@workspace /docs/kiwoom_api/주식 매수주문(kt10000).md 파일을 참고해서 
주식 매수 로직을 구현해줘
```

또는:

```
@workspace docs/kiwoom_api 디렉토리의 API 문서를 참고해서
잔고 조회 API를 호출하는 코드를 작성해줘
```

## 🔄 문서 업데이트

엑셀 파일이 업데이트되면 다음 스크립트로 마크다운을 재생성할 수 있습니다:

```bash
poetry run python convert_api_doc_to_md.py
```

## 📋 API 카테고리

주요 카테고리:

- **OAuth 인증** (2개): 토큰 발급/폐기
- **국내주식** (130+개): 시세, 차트, 투자자별 매매 등
- **계좌** (20+개): 잔고, 체결, 주문 등
- **주문** (10+개): 매수/매도/정정/취소
- **ELW** (10+개): 이론가, 민감도 등
- **ETF** (10+개): NAV, 시세 등
- **금현물** (15+개): 매매, 시세 등
- **실시간 시세** (20+개): WebSocket 구독
- **프로그램매매** (10+개): 추이, 누적 등
- **기타**: 업종, 테마, 조건검색 등

- `키움 REST API 문서.xlsx`: 원본 엑셀 파일
- `api/Kiwoom.py`: 실제 API 클라이언트 구현

## 💡 팁

### 자주 사용하는 API

1. **인증**: `접근토큰 발급(au10001).md`
2. **종목 정보**: `주식기본정보요청(ka10001).md`, `종목정보 리스트(ka10099).md`
3. **주문**: `주식 매수주문(kt10000).md`, `주식 매도주문(kt10001).md`
4. **잔고**: `계좌평가잔고내역요청(kt00018).md`
5. **시세**: `주식호가요청(ka10004).md`, `체결정보요청(ka10003).md`

### API ID 패턴

- `au*`: OAuth 인증
- `ka*`: 조회 (시세, 잔고, 계좌 등)
- `kt*`: 거래 (주문, 잔고 등)
- `0*`: 실시간 WebSocket 구독

## ⚠️ 주의사항
