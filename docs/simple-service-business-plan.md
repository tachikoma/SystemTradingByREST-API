# 주식 자동매매 서비스 기획안

현재 코드베이스를 기반으로 실전 운영 가능한 자동매매 서비스를 구축하기 위한 기획입니다.

---

## 1. 인프라 & 배포

### 1.1 서버 환경
- **클라우드 VM** (AWS EC2, GCP Compute Engine 등)
  - Ubuntu 22.04 LTS (안정성)
  - Python 3.11+, Poetry 설치
  - 24시간 실행 환경 (cron/systemd로 자동 재시작)
  
- **로컬 서버 대안**
  - Mac Mini / Raspberry Pi 4 (저전력)
  - UPS 백업 전원 (정전 대비)
  - 외부 접속용 고정 IP/DDNS

### 1.2 프로세스 관리
````bash
# systemd 서비스 등록 예시
# filepath: /etc/systemd/system/stocktrading.service
[Unit]
Description=Stock Trading Bot
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/home/ubuntu/SystemTrading
ExecStart=/home/ubuntu/.local/bin/poetry run python main.py --yes
Restart=always
RestartSec=30
Environment="KIWOOM_MODE=real"

[Install]
WantedBy=multi-user.target
````

## 3. 위험 관리

### 3.1 자금 관리
- **계좌 분리**
  - 전용 증권 계좌 (생활비와 분리)
  - 초기 자금: 최소 500만원 권장 (분산투자 위해)
  
- **일일 손실 한도** (추가 구현)
  - 하루 -5% 손실 시 거래 중단
  - 다음날 09시 자동 재개

### 3.2 긴급 중단 메커니즘
````python
# ...existing code...
def check_emergency_stop(self):
    """긴급 중단 조건 체크"""
    today_loss_rate = self.calculate_daily_pnl() / self.initial_deposit
    
    if today_loss_rate < -0.05:  # -5% 손실
        self.logger.error(f"긴급 중단: 오늘 손실률 {today_loss_rate:.2%}")
        self.send_telegram(f"⚠️ 긴급 중단! 손실률: {today_loss_rate:.2%}")
        self.stop()  # 전략 중단
        return True
    return False
````

## 5. 법적 & 컴플라이언스

### 5.1 세금 (한국 주식 기준)

> 본 시스템은 **한국 주식 (KOSPI/KOSDAQ)** 자동매매를 대상으로 합니다.

#### A. 한국 주식 세금 (2026년 기준)

**거래 단계 세금**:
- **증권거래세**: 0.20% (매도 시만 부과)
  - 코스피: 0.20% (2025년 0.18% → 2026년 0.20% 인상)
  - 코스닥: 0.20%
  - 매수 시에는 세금 없음
  - 자동 공제 (증권사가 체결 시 차감)

- **거래 수수료**: 0.015% (증권사별 상이)
  - 키움증권 실전 기준: 온라인 0.015%
  - 모의투자: 0.35% (실전보다 높음)

**소득세**:
- **양도소득세**: 일반 개인투자자는 **면세**
  - ⚠️ 대주주만 과세 (종목당 10억 이상 또는 지분 1% 이상)
  - 대주주 세율: 20% (3억 초과분 25%)
  - 본 시스템 유저는 해당 없음 (소액 투자자)

- **배당소득세**: 14% (지방소득세 1.4% 포함 시 15.4%)
  - 배당금 수령 시 자동 원천징수
  - 연간 금융소득 2,000만원 이하: 분리과세 (14%로 종결)
  - 2,000만원 초과: 종합소득세 대상 (6-45% 누진세율)

**실제 비용 계산 예시**:
````python
# 1,000만원 매도 시 실제 수령액
sell_amount = 10_000_000  # 매도금액
transaction_fee = sell_amount * 0.00015  # 거래수수료 1,500원
securities_tax = sell_amount * 0.0020    # 증권거래세 20,000원

net_received = sell_amount - transaction_fee - securities_tax
# 9,978,500원 수령 (약 0.215% 차감)
````

#### B. 미국 주식 세금 (참고용)

> 본 시스템은 미국 주식을 지원하지 않지만, 향후 확장 시 참고용으로 기재합니다.

**거래 단계**:
- **증권거래세**: 없음
- **거래 수수료**: 증권사별 상이 (보통 건당 고정 $0-5)

**소득세**:
- **양도소득세**: 
  - 연간 250만원까지 **비과세**
  - 250만원 초과분: 22% (지방소득세 2% 포함)
  - 예: 연간 500만원 수익 → (500-250) × 22% = 55만원 납부

- **배당소득세**:
  - 미국 원천징수: 15% (한미 조세조약)
  - 국내 종합소득세: 금융소득 2,000만원 초과 시 추가 과세

#### C. 세무 처리

**거래 내역 보관**:
- **보관 기간**: 5년 (전자상거래법)
- **보관 대상**: 
  - 체결 내역 (매수/매도 확인서)
  - 계좌 잔고 스냅샷 (월말 기준)
  - 손익 계산서 (연말)

**신고 의무**:
- **한국 주식**: 일반 투자자는 별도 신고 불필요
  - 증권거래세는 자동 납부
  - 배당소득세는 원천징수로 종결
- **대주주**: 양도소득세 신고 (매년 5월)
- **미국 주식**: 양도차익 250만원 초과 시 종합소득세 신고 (5월)
